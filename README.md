# Higher or Lower Game

> Guess which celebrity has more Instagram followers — keep your streak alive!

---

## Features
- 10 real celebrity follower counts
- Streak-based scoring
- One wrong guess ends the game
- Replay support

## Requirements
```
No external dependencies — pure Python 3
```

## Installation
```bash
git clone https://github.com/brainiacweb-tech/higher-lower-game
cd higher-lower-game
pip install -r requirements.txt
```

## Usage
```bash
python main.py
```
